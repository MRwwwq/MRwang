## FinSentiment引擎性能

### 架构
```
FinSentiment:
  ├── Layer 1: 金融关键词词典(80+正面 + 70+负面 + 12修饰词)
  ├── Layer 2: SnowNLP通用中文情感(辅助)
  └── Hybrid: 词典信号强→70%字典+30%SnowNLP
             无明确信号→40%字典+60%SnowNLP
```

### 与FinBERT对比

| 方案 | 状态 | 优势 | 劣势 |
|:----|:----|:----|:----|
| FinBERT(ProsusAI) | ❌ HuggingFace不可达 | 专业金融模型 | 需下载~500MB |
| FinBERT(中文版) | ❌ HuggingFace不可达 | 中文金融优化 | 需下载~500MB |

## 改进计划

| 改进项 | 优先级 | 方案 |
|:------|:-----|:----|
| 补词典: 套保/浮亏/公允价值等财务术语 | P0 | 追加至FIN_NEGATIVE |
| 词典权重调优 | P1 | 减少SnowNLP干扰,词典信号>0.15时权重提至85% |
| 新闻正文获取 | P2 | 通过`major_news`的URL爬取正文 |
| 机构调研(survey)权限 | P2 | 提升Tushare积分 |
| FinBERT离线模型 | P3 | 手动下载后导入容器 |
