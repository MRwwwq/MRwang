# 五阶段标准化投研分析SOP

> **版本:** v1.0 | **生效:** 2026-07-20
> **适用范围:** 所有个股AI投研分析，严格按此流程执行，不可跳步、不可篡改参数

---

## 第一阶段：前置准备（每次启动必做）

### 1.1 加载知识库RAG文档
- 加载skill `stock-analysis-framework`（全局规则/四层降级/诱多/资金库/五维/11模块）
- 加载skill `five-new-analysis-modules`（M1~M5五模块+训练案例）
- 加载标的对应训练案例（如600884→training_case_600884.md）
- 全量量化规则、资金判定阈值、芒格25误判风控案例已固化在FAISS

### 1.2 数据源状态确认
```
Tushare Pro → AkShare Sina → 雪球(302→EM替代) → 腾讯行情API
```
- Tushare: ✅ daily/moneyflow/margin/forecast/fina_indicator 全部可用
- 雪球: 🔴 自动化接口封禁(WAF挑战页面)，仅使用东方财富EM替代
- 降级链: `data_fallback.py` 四级自动降级

### 1.3 清空历史无关数据
- 单次仅分析1只标的（排他规则已固化）
- 清除上一只股票缓存数据
- 每日重置misjudge_25_factors判定（不延续昨日持仓/观点）

## 第三阶段：模型多层推理研判（核心）

### 3.1 赛道维度模型校验
- 判定规则：行业近一年景气增速>10%，限定AI算力/储能/半导体/新能源/高端制造
- 输出：赛道景气得分、行业排名，不达标直接拦截
- 实现：`weight_snapshots/*/industry_model.md` 行业模型映射

### 3.2 基本面模型校验
量化阈值自动比对：
| 指标 | 阈值 |
|:----|:-----|
| 连续2季扣非净利同比 | >0（为正） |
| 经营现金流 | >0（为正） |
| PE 52周分位 | 20%~70% |
| 市值 | 80~1500亿 |
| 大额商誉 | >净资产50%→标记高风险 |

### 3.3 产业链催化识别
- 检索政策/扩产/上下游大单催化事件
- 无正向催化降分
- 实现：`cross_compare.py` 五股横向对比

### 3.4 市场热点热度校验
- 近30日板块热度前30%（社会认同因子）
- 持续机构研报覆盖（forecast数据）
- 冷门无资金题材剔除

### 3.5 技术上升车道识别（右侧交易模型）
- 仅认可上升趋势标的（均线多头排列：MA5>MA10>MA20>MA60）
- 左侧下跌趋势直接过滤
- 叠加筹码成本分布（MA5/10/20/30/60五档）
- 实现：`rag_report_with_5modules.py` M1模块

### 专项资金子模型
统计近10日主力资金连续流入/流出行为，区分四种形态：
| 形态 | 判定条件 | 操作 |
|:----|:---------|:----|
| 温和建仓 | 连续小单流入+价格不涨 | 关注 |
| 放量拉升 | 量比>1.5+大单持续流入 | 可跟进 |
| 拉高诱多 | 涨停日+后续持续出货 | 🔴规避 |
| 高位出货 | 放量滞涨+大小单背离 | 🔴减仓 |

## 第四阶段：风控层二次拦截

### 4.1 高风险标的黑名单
- ST/*ST、退市股 → 自动拦截
- 单日换手>25% → 🔴标记
- 市值<50亿纯题材股 → 🔴标记
- 实现：`misjudge_25_factors.py` + `five_new_modules.py` M4黑天鹅筛查

### 4.2 心理误判对标（芒格25误判RAG）
- 自动计算25因子 → 高分(>=60)预警
- ≥3项高分 → Lollapalooza一票否决
- 匹配FAISS历史案例（29MD/112chunks/2110dim）
- 实现：`misjudge_25_factors.py` + `rag_report_with_5modules.py`

### 4.3 宏观/衍生品黑天鹅筛查
| 标的关联风险 | 筛查模块 |
|:------------|:--------|
| 金价波动(黄金股) | M5宏观联动 |
| 锂矿价格下行(储能/锂电) | M3产业隐藏变量 |
| 汇率冲击(海外业务) | M3产业隐藏变量+五模块M3 |
| 利率上行(重资产) | M5宏观联动(Shibor) |

### 4.4 完整风控打分表
- Lollapalooza ≥40分 → 禁止交易
- 5模块平均风险分（M1~M5）>50 → 降仓50%
- 实现：`rag_report_with_5modules.py` 板块4+板块5综合

统一输出11+5模块（原11模块 + 五大新增模块）：

| 模块 | 内容 | 源文件 |
|:----|:-----|:-------|

| ⑥ | 北向季度持股情况 | Tushare hk_hold |
| ⑦ | 技术形态右侧趋势判定 | M1高频盘口(RSI/KDJ/MACD) |
| ⑧ | 雪球散户多空/机构预期 | EM替代+forecast |
| ⑨ | 52周估值分位 | daily_basic PE分位 |
| ⑩ | 芒格25误判风险对标打分 | misjudge_25_factors.py |

| ⑫~⑯ | M1~M5五大新增模块 | five_new_modules.py |

## 硬性约束总结

| 约束 | 说明 |
|:----|:-----|
| 单标的排他 | 每次仅分析1只，禁止全市场批量 |
| 数据源降级链 | Tushare→AkShare→EM→腾讯，自动降级 |
| 四层数据源标记 | 所有表格尾部标注来源 |
| 误判每日重置 | 不延续昨日持仓/观点 |
| Lollapalooza一票否决 | ≥3项高分→禁止开仓加仓 |
| 左侧不交易 | 仅认可右侧上升趋势 |
| 雪球离线标注 | 人工采集数据单独备注 |
| 无高频自动交易 | 仅输出研判结论，不自动下单 |
